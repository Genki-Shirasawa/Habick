# ローカルでの動作方法
1. リポジトリをクローンする
```bash
git clone https://github.com/Genki-Shirasawa/Habick.git
```
2. リポジトリに移動
```bash
cd Habick
```
3. 仮想環境の構築
```bash

```
5. 必要なライブラリ等をインストールする
```bash
pip install -r requirement.txt
```
4. サーバーを立ち上げる
```
python manage.py runserver
```
